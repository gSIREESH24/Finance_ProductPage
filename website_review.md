### Q1: Identify 2 Specific Issues

**1. EMI Calculator Layout Issues:**  
The “EMI Calculator” heading is placed too close to the navigation bar, significantly reducing visual clarity. Additionally, the inputs and output results are stacked vertically. This makes the page unnecessarily long; shifting to a side-by-side (left-right) layout would drastically improve readability and space utilization.

**2. Congested Text in the "About Us" Section:**  
The "Meet RupeeVyze" section suffers from poor text arrangement. The paragraphs are very congested and packed closely together. Breaking up the long blocks of text with bullet points, increasing the line height (leading), or adding more whitespace between paragraphs would make this section much easier for users to scan and digest.

### Q2: Which One Would You Fix First & Why?

**I would fix the EMI Calculator layout issue first.**  
Arranging calculators side-by-side directly impacts the core functionality of the product. When inputs and outputs are immediately visible next to each other, usability skyrockets because users can instantly see how changing a value affects their results without needing to scroll.
